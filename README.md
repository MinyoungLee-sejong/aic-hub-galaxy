# AIC Hub Galaxy

A configurable, galaxy-themed frontend shell built with React, TypeScript, and React Three Fiber.

## Live site

The public site is deployed with GitHub Pages at:

`https://minyounglee-sejong.github.io/aic-hub-galaxy/`

## Run locally

```bash
npm install
npm run dev
```

Open [http://127.0.0.1:5173](http://127.0.0.1:5173). Use the profile button in the top-right corner to configure the `Try now` destination and sidebar menu links.

## Verify

```bash
npm test
npm run lint
npm run build
```
